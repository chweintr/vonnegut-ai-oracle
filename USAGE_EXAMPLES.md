- "If you were teaching this to high school students, what would you want them to understand?"

**Step 4:** Broader Themes
- "How does your anti-war stance appear in even your non-war stories?"

## Tips for Effective Use

### Do's ‚úÖ
- **Be specific:** Copy exact passages for detailed analysis
- **Ask follow-ups:** Conversations build deeper understanding
- **Mix modes:** Use both Chat and Reading Guide for complete experience
- **Take notes:** Export or copy important insights
- **Experiment:** Try different types of questions
- **Context matters:** Provide context for best responses

### Don'ts ‚ùå
- **Don't expect perfection:** This is AI, not the real Vonnegut
- **Don't plagiarize:** Use for learning, not copying into essays
- **Don't skip reading:** AI supplements, doesn't replace actual reading
- **Don't ignore disclaimers:** Remember this is simulation
- **Don't overload:** Focus on one passage or theme at a time

### Best Practices

**For Students:**
- Use as a study supplement, not primary source
- Verify important claims with scholarly sources
- Cite as "AI simulation" if referencing in papers
- Discuss insights with teachers and classmates

**For Teachers:**
- Demo in class to show interactive literature study
- Create guided question sets for students
- Use for differentiated learning support
- Encourage critical thinking about AI limitations

**For Writers:**
- Use for craft discussions and technique analysis
- Don't copy style slavishly - find your own voice
- Appreciate Vonnegut's approach, then diverge
- Use feedback as starting point for revision

**For Book Clubs:**
- Prepare questions in advance
- Share interesting responses with group
- Use to jumpstart discussions, not replace them
- Encourage diverse interpretations

## Troubleshooting Common Issues

**"The AI doesn't sound like Vonnegut"**
- Educational mode is slightly more formal for teaching
- Try Chat mode for more authentic conversational voice
- Provide more context about what you're asking

**"Responses are too general"**
- Copy specific passages for analysis
- Ask more focused questions
- Provide context from the story

**"I want deeper analysis"**
- Ask follow-up questions
- Request specific literary techniques
- Ask about connections to other works or biography

**"Voice isn't working"**
- Check that ElevenLabs API keys are set
- Try Text‚ÜíAudio mode first
- Verify API limits haven't been reached
